# SmartNotes API
SmartNotes API is a lightweight backend built with Node.js, Express, and MongoDB. It provides simple RESTful endpoints for creating, reading, updating, and deleting notes.

## Tech Stack
- React
- Node.js, Express
- MongoDB, Mongoose
- Hosted on GitHub Pages and Render

## Features
- CRUD operations for notes
- RESTful API structure
- MongoDB + Mongoose integration
- Simple and clean codebase
